## 🔴 משימות קריטיות - עדיפות גבוהה

### 1. **יצירת משתמש חדש (Create User)**
**סטטוס:** ❌ חסר לחלוטין

**מה צריך:**
- טופס ליצירת משתמש חדש ב-`UsersManagement.tsx`
- שדות נדרשים:
  - אימייל (חובה)
  - שם (אופציונלי)
  - בחירת פרופיל (dropdown מרשימת הפרופילים)
  - בחירת מצב: BYOK / MANAGED
  - סטטוס: פעיל/לא פעיל
- Modal/Dialog ליצירת משתמש (כמו בפרופילים)
- הצגת ה-API Key שנוצר למשתמש **פעם אחת בלבד** אחרי היצירה
- אזהרה למשתמש לשמור את המפתח (לא ניתן לשחזר)

**API קיים:**
```
POST /api/users
Body: { email, name?, profileId?, mode, isActive }
Response: { success, user, proxyApiKey }
```

**חשיבות:** 🔥 קריטי - בלי זה לא ניתן ליצור משתמשים חדשים

### 2. **עריכת משתמש (Edit User)**
**סטטוס:** ❌ חסר לחלוטין

**מה צריך:**
- כפתור "ערוך" בכל כרטיס משתמש
- Modal לעריכה עם השדות:
  - שם
  - פרופיל (שינוי פרופיל)
  - מצב (BYOK/MANAGED)
  - סטטוס (פעיל/לא פעיל)
- **אסור** לאפשר שינוי אימייל (unique identifier)
- שמירת השינויים וריענון הרשימה

**API קיים:**
```
PUT /api/users/:id
Body: { name?, profileId?, mode?, isActive? }
```

**חשיבות:** 🔥 קריטי - צריך לעדכן משתמשים קיימים

### 4. **קישור משתמש לפרופיל (User-Profile Assignment)**
**סטטוס:** ⚠️ קיים חלקית - רק הצגה, אין עריכה

**מה צריך:**
- הצגת שם הפרופיל (לא רק ID) בכרטיס המשתמש
- Dropdown לבחירת פרופיל בעת יצירה/עריכה
- אפשרות להסיר פרופיל (לא חובה)
- אינדיקציה ויזואלית אם למשתמש אין פרופיל

**שיפור נוסף:**
```typescript
// צריך לטעון את רשימת הפרופילים
const [profiles, setProfiles] = useState<Profile[]>([]);

// ולהציג בשם במקום ID
const profileName = profiles.find(p => p._id === user.profileId)?.name || 'ללא פרופיל';
```

**חשיבות:** 🔥 קריטי - הקשר בין משתמש לפרופיל הוא הליבה של המערכת

```typescript
// יצירת משתמש
const response = await apiCall(API_ENDPOINTS.users, {
  method: "POST",
  body: JSON.stringify({
    email: "user@example.com",
    name: "John Doe",
    profileId: "profile_id_here",
    mode: "BYOK",
    isActive: true
  })
});

// שמירת ה-API Key שחוזר
const { proxyApiKey } = response;
// הצג למשתמש פעם אחת!
```

### טעינת פרופילים ל-Dropdown
```typescript
useEffect(() => {
  const loadProfiles = async () => {
    const data = await apiCall<Profile[]>(API_ENDPOINTS.profiles);
    setProfiles(data);
  };
  loadProfiles();
}, []);
```

```typescript
const [editingUser, setEditingUser] = useState<User | null>(null);

// פתיחת modal
<button onClick={() => setEditingUser(user)}>ערוך</button>

// ב-modal
{editingUser && (
  <Modal onClose={() => setEditingUser(null)}>
    <form onSubmit={handleUpdate}>
      {/* טופס עריכה */}
    </form>
  </Modal>
)}
```
